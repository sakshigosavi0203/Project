ECOMMERCE SALES : 

Project Overview
This project is an interactive PowerBI dashboards built to analyze e-commerce sales performance.  
It provides insights into sales, profit, quantity, average order value, customer behavior, and product performance.

 Objectives
- Track overall Sales, Profit, Quantity, and AOV
- Analyze state-wise sales performance
- Understand monthly profit trends
- Identify top customers, categories, and sub-categories
- Compare payment modes used by customers

Dataset Details
The dataset contains:
- Order details
- Customer information
- Product categories & sub-categories
- Payment modes
- Sales, profit, and quantity metrics

 Tools & Technologies
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Microsoft Excel(data source)

 Dashboard Features
- KPI Cards: Total Sales, Profit, Quantity, AOV
- Sales by State
- Profit trend by Month
- Quantity by Category & Payment Mode
- Profit by Sub-Category
- Interactive Quarter & State filters
